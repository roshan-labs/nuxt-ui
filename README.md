# @roshan-labs/ui

![npm](https://img.shields.io/npm/v/@roshan-labs/ui?color=%2342b883)

封装 `element-plus` 组件库自动导入 `Nuxt`

## 安装

```bash
pnpm add @roshan-labs/ui
```

## 使用

```typescript
export default defineNuxtConfig({
  modules: ['@roshan-labs/ui'],
  ui: {
    // 选项
  },
})
```
